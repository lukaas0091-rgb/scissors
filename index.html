TikTok LIVE Key

Vincula tu sesión TikTok, genera la stream key para OBS y dispara el botón Go LIVE desde un solo panel.

Esperando acción.
  1. Presiona Vincular TikTok.
  2. Completa el login en la sesión Puppeteer si tu entorno lo permite.
  3. Presiona Generar Stream Key para extraer server y key.
  4. Pega la key en OBS y luego usa Iniciar LIVE.

Este MVP mantiene un único navegador Puppeteer abierto en memoria. En Render Free la sesión se conserva mientras el contenedor siga activo.
